1. RDF and Python

Given the following data in Turtle notation: 

   @prefix schema: <http://schema.org/> .

   <http://rdfa.info/play/#oswald> a schema:Person ;
       schema:givenName "Oswald" ;
       schema:familyName "Muster" ;
       schema:email "Oswald.Muster@mq.edu.au".

Store this data in a file with the name example.ttl. Write a Python script that uses the rdflib library. This Python script should first parse the data into a graph and then display the following information:

  1. the number of statements in the graph;
  2. the entire graph serialized in RDF/XML notation;
  3. the subject, predicate, and object of each statement in this graph.

The output of your Python script should look similar to the output here:

3. Querying an RDFa-augmented XHTML page via Python/SPARQL

Below is a simple XHTML document that has been augmented with RDFa markup: 

<!DOCTYPE html>  
   <html
      xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en" 
      prefix="dc: http://purl.org/dc/elements/1.1/">
     <head>
       <title>Ronald McDonald's Home Page</title>
       <link rel="profile" href="http://www.w3.org/1999/xhtml/vocab" />
       <base href="http://localhost:8080/ronald-mcdonald/" />
       <meta property="dc:creator" content="Ronald McDonald" />
       <link rel="foaf:primaryTopic" href="http://localhost:8080/ronald-mcdonald/#me" />
     </head>
     <body about="http://localhost:8080/ronald-mcdonald/#me">
       <h1>Ronald McDonald's Home Page</h1>
       <p>Hi, my nickname is <span property="foaf:nick">Mickey D.</span> and I like
         <a href="http://www.kfc.com.au/" rel="foaf:interest"  
           lang="en">KFC Australia</a>.
       </p>
       <p>
         My <span rel="foaf:interest" resource="urn:ISBN:071815780X">favorite
         book is <span about="urn:ISBN:071815780X"><strong
         property="dc:title">Jamie's 15-Minute Meals</strong> by
         <span property="dc:creator">Jamie Oliver</span></span></span>.
       </p>
     </body>
   </html>

Save this document as index.html in a folder named ronald-mcdonald. From a level above this folder, launch a Python HTTP server: 

  c:>python -m http.server 8080

and access the web page via http://localhost:8080/ronald-mcdonald/ and you should see the following in Google Chrome:




Now write a Python script that uses the rdflib 6.1.1 library (pip install rdflib) that relies on PyRDFA (pip install pyRdfa3) for RDFa processing. The Python script should first fetch the XHTML page and then parse it into an RDF graph, and then extract the following things:

  1. the RDFa from the XHTML page and displays this information as RDF/XML;
  2. the name of the creator of the Web page and displays this information in form of a JSON object;
  3. the primary topic of the Web page and displays this information in form of a JSON object;
  4. the nick name of the person that this Web page describes and displays this information in form of a JSON object;
  5. the interests of the person that this Web page describes and displays all information in form of a JSON object;
  6. the book (title) that Mickey D. is interested in and displays this information in form of a JSON object.

You can find some hints on how to solve this task in the lecture notes, but for some details you will probably also have to check the rdflib documentation or do a Google search. Note that the output of your Python script should look as follows:
